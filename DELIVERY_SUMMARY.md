## 🚀 Deployment - 3 Easy Options

### Option 1: GitHub Pages (5 minutes)
```bash
# 1. Create GitHub repo "datarn-website"
# 2. From directory: git init, git add ., git commit, git push
# 3. GitHub Settings → Pages → Deploy from /out folder
# Result: Free hosting, auto-updates with each push
```

### Option 2: Netlify (3 minutes)
```bash
# 1. Visit netlify.com
# 2. "New site from Git" → Select repo
# 3. Build: npm run export, Publish: out/
# Result: Free domain, automatic deployments
```

### Option 3: Vercel (1 minute - EASIEST)
```bash
# 1. Visit vercel.com
# 2. Import GitHub project
# 3. Click "Deploy"
# Result: Optimized for Next.js, free tier
```

## 💡 Customization Points

All customization is simple and documented:

- Reference in components
