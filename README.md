# Datarn - High-Performance Python Enhancement Platform

A modern, static website for the Datarn project - an open-source platform providing Rust+Python and Go+Python custom libraries to enhance Python development.

## 🚀 About Datarn

**Datarn** is dedicated to overcoming Python's inherent limitations while maintaining elegant simplicity. We combine:

- **Python**: Simplicity, readability, and rapid development
- **Rust**: Memory safety, performance, and reliability
- **Go**: Built-in concurrency, fast compilation, and easy deployment

Visit the website to explore our libraries and join the community!

## 🌐 Technologies Used

- **Next.js 14**: Modern React framework with static export
- **Tailwind CSS**: Utility-first CSS framework for styling
- **TypeScript**: Type-safe JavaScript
- **Responsive Design**: Mobile-first approach

## 🚀 Getting Started

### Prerequisites

- Node.js 16+
- npm or yarn

### Installation

```bash
cd datarn-website
npm install
```

### Development

```bash
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to see the site in development mode.

### Build for Production

```bash
npm run build
```

### Static Export

```bash
npm run export
```

This generates static HTML files in the `out/` directory that can be deployed to any static hosting service.
