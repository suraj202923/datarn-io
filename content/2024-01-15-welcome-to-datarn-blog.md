## 💡 Markdown Formatting Guide

### Headers
```markdown
# Main Title (H1)
## Section Title (H2)
### Subsection (H3)
```

### Code Blocks
```python
# Python example
def optimize_with_datarn():
    return "Lightning fast! ⚡"
```

```rust
// Rust example
fn datarn_power() -> u32 {
    1000
}
```

### Lists & Emphasis
- **Bold text** for important concepts: `**bold**`
- *Italic text* for subtle emphasis: `*italic*`
- Mix them: ***bold and italic***
